# DeepCausality Effects — DEPRECATED

This crate is no longer maintained. 

The functionality it provided was superseded
by the propagating-effect model in `deep_causality`'s effect propagation
process, and the types in this crate are no longer used by any
DeepCausality crate.

The published versions on crates.io (0.0.1 – 0.0.7) remain available
for any existing dependents, but no further releases are planned.

**The crate name `deep_causality_effects` is effectively free for
reuse.** If you wish to take it over, contact the DeepCausality
maintainers via the repository linked below.

Repository: https://github.com/deepcausality/deep_causality.rs
