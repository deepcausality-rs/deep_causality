The [corridor](/examples/plasma-blackout-corridor/) flies one descent. This
example flies the same descent across six atmospheres and reduces the result to a
lookup table with error bars.

The counterfactual axis differs. The corridor forks a paused state and varies a
command. This example varies the world: six atmospheres alternated from one
validated baseline, each carrying its own `!!ContextAlternation!!` provenance.

## The table

```text
  world          dT K   rho    IMU dep  onset s  exit s  dwell s   drift(dark) m
  standard_day      0   1.00     1.00     11.9    70.0     58.2    45.93 +- 2.35
  polar_winter    -40   1.20     1.40      9.9    69.8     60.0    68.75 +- 5.19
  dense_day         5   1.30     1.05      8.7    69.7     61.1    52.03 +- 5.73
  hot_day          ...                                            53.16 +- 2.38
  cold_day         ...                                            57.72 +- 2.05
  thin_day         ...                                            45.72 +- 1.96
```

Each row is eight full descents with deterministic receiver-noise draws, so the
`±` is a real ensemble spread rather than a guess. Forty-eight descents total.

## Measured result

```text
[PASS] (4) the INS does not behave as assumed in the cold: polar-winter mean
       blackout drift 68.75 m vs standard-day 45.93 m (1.50x; gate requires 1.2x)

[PASS] (4b) the cold effect is statistically resolved: polar-standard separation
       22.81 m vs combined sigma 5.69 m (4.0 sigma; gate requires 2)
```

A cold, dense atmosphere ionizes earlier and holds the sheath longer, so the
vehicle spends more time dead-reckoning. Drift increases by 50%.

Gate 4b establishes that the separation exceeds the ensemble spread. Without the
per-row error bars, the 1.50× ratio would rest on two single runs.

The example also cross-checks the mechanism against a closed form:

```text
predicted = 45.93 * departure * (dwell/58.2)^2
          = 68.3   vs measured 68.75 +- 5.19
```

Free inertial propagation gives drift proportional to elapsed time squared. The
measured means agree with that form to within about 5%.

## Intended use

The output is a lookup table for onboard use. Given a measured temperature
departure, it returns the expected blackout window and navigation dispersion for
margin sizing.

The Shuttle programme applied the same pattern on the ascent side, regenerating
guidance I-loads on the day of launch from measured balloon soundings.

## Audit trail

Forty-eight per-branch audit logs and one main log are written to
`weather/audit/`. Each names the baseline its world was alternated from, so every
table row is traceable to the run that produced it.

## Applicability limits

The physics is validated at one flight anchor, RAM-C II. The five off-anchor rows
are single-anchor extrapolation and are not independent validations.

The parameter set covers temperature and density. Environmental qualification of
the DO-160 type additionally covers humidity, winds and statistical coverage.

The table is one artifact in a certification chain.
