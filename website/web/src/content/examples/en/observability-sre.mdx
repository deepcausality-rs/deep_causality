## The problem

A user-facing 5xx never tells the whole story. Underneath it is queue backlog, retry storms, GC pauses, slow database queries, and a dozen other suspects. The page that pinged the on-call engineer fired *after* the chain had already played out. You want to look at the chain itself.

## The shape of the code

Each suspect is a Causaloid that evaluates against an `IncidentWindow`. Compose them into a chain. Reason *backward* from the symptom to find the originating event.

```rust
use deep_causality::*;

let db_lag       = Causaloid::new(|w: &IncidentWindow| Ok(w.db_p99 > 250));
let queue_full   = Causaloid::new(|w: &IncidentWindow| Ok(w.queue_depth > w.queue_capacity * 0.85));
let retry_storm  = Causaloid::new(|w: &IncidentWindow| Ok(w.retries_per_sec > w.baseline_rps * 4.0));
let frontend_5xx = Causaloid::new(|w: &IncidentWindow| Ok(w.error_rate > 0.05));

let chain = Causaloid::compose(&[db_lag, queue_full, retry_storm, frontend_5xx]);

let root = chain
    .reason_back(&incident_window)?
    .first_cause()
    .ok_or_else(|| anyhow!("no cause identified inside window"))?;

println!("root cause: {}", root.label());
```

## Walkthrough

`Causaloid::compose` orders the rules. The chain is read as "A leads to B leads to C leads to D", and `reason_back` walks that order in reverse.

`reason_back` returns a `CausalTrace`: every node it visited, the time at which the node's predicate first held inside the window, and the supporting evidence. `first_cause` is the earliest node in the trace whose predicate held. That node is the root.

The trace is the thing you actually want in a post-mortem. It is the audit log the page never gave you.

## Run it

```bash
cargo new --bin rca
cd rca
cargo add deep_causality
# wire `IncidentWindow` to your metrics warehouse adapter and run on a fixture incident
cargo run --release
```

## Why this is a good fit

You can add a new suspect in one place. You can replay an incident offline with a recorded window and verify that the chain still points at the right node. You stop arguing about which dashboard told the truth.
